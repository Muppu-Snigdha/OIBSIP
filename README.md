## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on Airbnb listing data using Python.  
The main goal of this project is to analyze customer preferences, room types, pricing trends, reviews, and availability patterns.

The project also includes:
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Standardization
- Outlier Detection
- Data Visualization


# 📂 Dataset Information

The dataset contains Airbnb listing information such as:

- Listing Name
- Host Details
- Neighbourhood Group
- Room Type
- Price
- Number of Reviews
- Availability
- Latitude & Longitude


# 🛠️ Technologies Used

- Python 🐍
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn


# 📊 Data Cleaning Tasks Performed

✅ Removed duplicate records  
✅ Handled missing values  
✅ Applied data standardization  
✅ Performed outlier detection  
✅ Analyzed pricing and reviews  


# 📈 Visualizations Included

The project contains the following graphs:

1. Neighbourhood Group Distribution
2. Price Distribution Histogram
3. Room Type Pie Chart
4. Reviews vs Price Scatter Plot
5. Availability Analysis Box Plot
6. Outlier Detection Graph
7. Average Price by Neighbourhood Group

# ▶️ How to Run the Project

## Step 1: Install Required Libraries


pip install pandas matplotlib seaborn scikit-learn

Step 2: Add Dataset

Place the dataset file inside the project folder and rename it as:

data.csv

Step 3: Run the Python File

python Data_cleaning.py

The project generates multiple graphical visualizations for analyzing Airbnb listings and customer trends.
